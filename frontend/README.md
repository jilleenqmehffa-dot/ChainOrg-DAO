# ChainOrg-DAO 前端

这是 ChainOrg-DAO 项目的前端组件，基于 React 和 Vite 构建。

## 开始使用

首先克隆仓库：

```bash
git clone <your-repository-url>
cd ChainOrg-DAO/frontend
```

安装依赖：

```bash
npm install
```

启动开发服务器：

```bash
npm run dev
```

## 项目结构

- `src/App.jsx` - 主应用组件
- `src/components/` - UI 组件
- `src/services/` - API 服务
- `src/index.css` - 样式文件

## 当前功能

目前项目实现了用户注册功能，可以通过 API 与后端通信。
